## MCU I2C Interface

Signal

- SA0 (Slave address Bit)
- SDA (I2C-bus Data Singnal)
  - Must be connected pull-up register
- SCL (I2C-bus Clock Signal)
  - Must be connected pull-up register
- RES#
  - used for the initialization of device
